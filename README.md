# 🚀 HibaVonal WebApp

This project is designed for dormitory students to report unexpected issues through the simplest process possible, while providing maintenance workers with the tools to manage requests and improve student life.

### 🛠️ Technologies
- **Frontend:** Blazor WASM (WebAssembly)
- **Backend Runtime:** .NET 8.0
- **Database:** SQLite + EF Core (Lightweight & Portable)
- **UI Library:** MudBlazor

---

### 📥 Quick Start

To get the project running locally, follow these steps:

1. **Clone the repository**

```bash
git clone <repo-url>
```

2. **Load dependencies**

```bash
dotnet restore
```

3. **Create database schema**

```bash
dotnet ef database update
```
